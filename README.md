**Matrix-Console**  
一个学生制作的简易矩阵计算器……（我只是个大一新生……）我的英语不太好……抱歉……

**目前已实现的主要功能**  
*   执行基本的矩阵计算（例如交换行、交换列等）。
*   允许用户保存和加载矩阵。
*   支持分数和整数。

**未来计划添加的功能**  
*   更多计算功能（例如计算秩、行列式、矩阵幂等）。
*   支持 `matrix_cat`（将一个矩阵合并到另一个矩阵的指定方向）。
*   如果可能，我们将支持从本地文件读取命令（这很难，因为我还不知道怎么做……）。

**主要开发思路**  
为了避免使用大量的 `if-else` 语句，我采用了一个“命令”结构体。这样，所有命令都将通过统一的方式调用。此外，这也使得添加新功能变得更加容易，尤其是在有多名开发者参与的情况下。你只需提交一个 .c 文件，就可以添加任何你想要的功能。（更多信息请阅读 `head.c` 和 `function.c`。）
